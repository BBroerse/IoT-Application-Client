# dashboard

## Configurations

Go to config/index.js and change the BASE_API
```
BASE_API: "THE_URL_TO_YOUR_API"
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
